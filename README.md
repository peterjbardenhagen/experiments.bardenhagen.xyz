## Projects

### 🏠 [Housing Market Predictor](./housing-predictor/)

Advanced AI-powered housing market analysis and prediction engine for Brisbane and Australia (2026-2050).

**Features:**
- Interactive market parameter controls (location, property type, budget, timeline)
- AI-powered property value forecasting (1-25 year periods) 
- Comprehensive risk assessment and ROI analysis
- Real-time dynamic charts and data visualization
- Mobile-first responsive design with dark theme

**Tech Stack:** HTML5, Tailwind CSS v4, Chart.js
